# NearHire

AI-powered multilingual hiring platform with integrated Candidate App, Employer Portal, and Admin Dashboard.

## Setup

1. Install dependencies
   ```bash
   npm install
   npm --workspace frontend install
   npm --workspace backend install
   ```

2. Run local development
   ```bash
   npm run dev
   ```

3. Start with Docker
   ```bash
   docker compose up --build
   ```

## Features

- Candidate nearby job discovery, OTP login, voice apply, WhatsApp quick apply
- Employer candidate pipeline, AI match scoring, interview scheduling, analytics
- Admin platform analytics, subscription management, WhatsApp API management
- AI workflows for job matching, resume parsing, multilingual voice transcription
